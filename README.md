# React.js Boostrap Template

An React application made with Bootstrap 4.

---

## Stack

| Lib | Version |
| ------ | ------ |
| react | 16.12 |
| react-dom | 16.12 |
| react-scripts | 3.3 |
| react-router-dom | 5.1 |
| react-redux | 7.1 |
| redux | 4.0 |
| fontawesome | 5.11 |
| bootstrap | 4.4 |
| node-sass | 4.13 |

---

## Running

**After clone**

```sh
$ cd react-components
$ npm install
```

**Starting the application**

```sh
$ npm start
```
